# ⚡ Low Consumption Consumer Tracker

Production-ready open-source desktop application for electricity utilities to identify low-consumption industrial consumers, support field inspections, and reduce system loss.

## Features
- Multi-month Excel merging
- Office-wise report generation
- Low-consumption detection
- Configurable thresholds
- Formatted Excel reports
- Desktop GUI (Tkinter)

## Quick Start

```bash
pip install -r requirements.txt
python merge_app.py
```

## Architecture

Input Excel Files → Validation → Merge → Office Segregation → Filtering → Excel Reports
